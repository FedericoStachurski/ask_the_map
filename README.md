# Ask the Map

Multimodal search over geolocated citizen science data (CommuniMap).

## Features (planned)
- Text + image embedding (CLIP)
- Multimodal search (FAISS)
- Fusion strategies (weighted, RRF)
- Map visualisation (Folium)

## Structure
- `src/ask_the_map/` → core library
- `scripts/` → runnable tools
- `data/` → raw data (ignored)
- `outputs/` → embeddings + maps (ignored)
- `models/` → embeddings models (ignored)
